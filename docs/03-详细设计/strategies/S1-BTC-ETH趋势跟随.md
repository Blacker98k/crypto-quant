# S1 · BTC/ETH 趋势跟随

> **风格**：趋势跟随（Trend Following）
> **状态**：规划阶段（Phase 3 实现）
> **预期角色**：核心仓位策略，吃 BTC/ETH 主升浪
> **最后更新**：2026-04-29

---

## 1. 策略思路

加密货币市场长期呈现**强趋势特征**——一旦 BTC/ETH 进入主升或主降阶段，趋势可持续数周至数月。趋势跟随策略的核心是：

> **承担 60% 的胜率亏损**，换取每次盈利时**远大于亏损**的盈亏比。典型 R:R = 2.5:1 以上。

不预测顶部和底部，只在**趋势确认后跟进**，趋势反转时果断离场。

## 2. 标的与周期

| 项 | 值 |
|---|---|
| 标的 | BTCUSDT, ETHUSDT（U 本位永续合约） |
| 不要标的 | 山寨币（流动性差、假突破多） |
| 入场判断周期 | **4h K 线** |
| 趋势过滤周期 | **1d K 线** |
| 平均持仓时长 | 3-30 天 |
| 杠杆 | **1-2x**（核心仓位，不激进） |

## 3. 入场条件

**全部满足才入场**：

### 3.1 大趋势过滤（1d K 线）

- **多头入场**：1d 收盘价 > 1d MA50 AND 1d MA20 > 1d MA50（金叉持续）
- **空头入场**：反向

→ 防止逆势开仓。

### 3.2 入场触发（4h K 线）

- **多头**：4h 收盘价突破最近 20 根 4h K 线的最高价（Donchian 通道上轨）
- **空头**：4h 收盘价跌破最近 20 根 4h K 线的最低价

### 3.3 波动率过滤

- 当前 ATR(14) > 历史 30 天 ATR 中位数
- 避免在低波动盘整期"假突破"

### 3.4 流动性过滤

- 该 4h K 线成交额 > 过去 7 天均值 × 1.2
- 过滤掉无量假突破

---

## 4. 出场条件

### 4.1 止损（强制交易所侧）

- **跟踪止损**：max(入场价 - 1.5 × ATR(14), 当前止损价) （多头）
- 每根 4h 收盘后**重新挂止损单**（先撤后挂）
- 止损必须挂在 Binance 侧（[ADR-003](../../07-决策日志/ADR-003-强制交易所侧止损.md)）

### 4.2 趋势反转出场（主动平仓）

- 4h 收盘价跌破 4h MA20（多头）
- 或 1d MA20 转跌破 1d MA50（大趋势失效）

### 4.3 时间出场

- 持仓超过 30 天且未达到 2R 盈利 → 平仓
- 防止"温水煮青蛙"式的小幅震荡

### 4.4 止盈

- **不设固定止盈**（趋势策略要让利润奔跑）
- 仅靠跟踪止损被动获利

## 5. 仓位管理

### 5.1 单笔仓位

```
风险金额 = 总资金 × 1%      （单笔风险 1%，[ADR-008 L2 层]）
仓位价值 = 风险金额 / (止损距离 / 入场价)
合约张数 = 仓位价值 / 入场价
```

### 5.2 加仓规则

- **首仓 50%**，等趋势确认后加仓
- 趋势确认条件：浮盈 ≥ 1R 后，4h K 线新高再加 25%
- 第 3 次加仓 25%，浮盈 ≥ 2R 时
- **加仓后止损统一上移**（保护利润）
- 总仓位上限：单标的 ≤ 25% 总资金（[ADR-008 L3 层]）

### 5.3 杠杆控制

- 名义杠杆 1-2x
- 加仓不增加名义杠杆（用浮盈做新仓位的保证金）

## 6. 风控规则（特定于本策略）

| 规则 | 阈值 | 动作 |
|---|---|---|
| 单策略单日亏 | ≥ 3% 总资金 | 暂停 24h |
| 单策略连亏 | ≥ 4 笔 | 暂停 24h |
| 单策略最大回撤 | ≥ 15% 策略资金 | 暂停 + 复盘 |
| 单标的同时持仓 | 1（不允许同时多 BTC 和空 BTC） | - |

策略级风控之上，还有 [ADR-008 三层风控](../../07-决策日志/ADR-008-三层风控冗余.md)兜底。

## 7. 预期表现（基于通用经验，待回测验证）

| 指标 | 目标 |
|---|---|
| 胜率 | 35-40% |
| 盈亏比 (R:R) | 2.5+ |
| 年化 | 30-40% |
| Sharpe | 1.0-1.5 |
| 最大回撤 | 15-20% |
| 月度盈利月数 | 6-7/12 |

**特征**：
- 多数月份小亏小赚
- 少数月份大盈利（吃到趋势）
- 震荡市最难受

---

## 8. 失效场景

哪些情况这个策略会亏：

1. **持续震荡市**（无趋势）→ 止损反复触发
2. **快速反转**（黑天鹅）→ 跟踪止损来不及调整
3. **假突破频繁**（流动性差时段）→ 入场后立即反向
4. **高 funding 期**（多头拥挤）→ 资金费持续吃利润

**应对**：
- 与 S2（均值回归）天然反相关，组合后平滑
- 与 S3（资金费率套利）独立
- 黑天鹅靠交易所侧止损 + 杠杆 ≤ 2x 兜底
